# Buddie IQ

Frontend for Buddie IQ, “the ring for plants, hives, and small habitats” — a zero‑battery, wireless environmental monitoring experience.

## Getting started

Prerequisites:

- Node.js 18+ and npm (recommend installing via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

Install and run locally:

```sh
git clone <YOUR_GIT_URL>
cd Buddie_IQ
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build for production with:

```sh
npm run build
```

Serve the contents of `dist/` with your preferred static hosting provider.
